    Vocab Definitions

    1. HTTP
      • Definition: Rules for transferring files to the internet
      • Source Doc
    2. HTML
      • Definition: Standardized system to tag files with text to communicate font, color, graphic, and hyperlink effects on World Wide Web pages
      • Source Doc
    3. Element
      • Definition: Combination of tags and data that tells our web browsers how to display information and content.
      • Source Doc
    4. Tag
      • Definition: Angle brackets used to create an element.
      • Source Doc
    5. Block (CSS)
      • Definition: Forms a visible block on the page, seperates into a new line from the previous content.
      • Source Doc
    6. Doctype
      • Definition: Required at top of all documents to ensure the internet does not convert the page to something different.
      • Source Doc
    7. Head
      • Definition: Contains all data for page that is not content for the viewers.
      • Source Doc
    8. Attribute
      • Definition: Extends a tag, changing or altering what the tag does.
      • Source Doc
    9. Boolean
      • Definition: Datatype that can only be true or false.
      • Source Doc
    10. DOM
      • Definition: Map that ends up explaining how different elements and attributes work together, referred to as parent, children and siblings.
      • Source Doc
    11. CSS
      • Definition: Language that shows how webpage looks.
      • Source Doc
    12. Property (CSS)
      • Definition: Two parts of the CSS rule set that helps set decide what will be changed and how the rule is going to change it.
      • Source Doc
    13. Selector (CSS)
      • Definition: What is being effected by the CSS rule.
      • Source Doc
